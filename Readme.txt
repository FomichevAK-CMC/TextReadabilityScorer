Анализ удобочитаемости русскоязычного текста с использованием различных индексов.

-----
Версия питона при разработке: 3.12 (должна работать и на 3.10, но нет 100% гарантии)
Зависимостей от внешних библиотек - нет.
-----

Аргументы командной строки:
    -f <path_to_file> [<encoding>] - передает указанный файл и кодировку(windows-1251 если не указана) анализатору.
    -t - запускает тесты.
    Если параметров нет - запрашивает пользователя ввести путь к файлу и кодировку.


Программа состоит из 4 файлов:
    - main.py - основной, исполняемый файл.
    - ru_analyzer.py - модуль с реализацией класса RUAnalyzer, в котором собраны методы для вычисления различных статистик и индексов удобочитаемости текста.
    - tscanner.py - модуль с реализацией настраиваемого токенизатора (функция scan_file), на основе регулярных выражений, который используется в модуле ru_analyzer.
    - tester.py - файл с функцией тестирования.

Папка tests должна находиться в одной директории с main.py
